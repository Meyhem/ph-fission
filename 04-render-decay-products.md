# 04-render-decay-products.md

## Overview
Based on decays from data, render products below main nucleus. Different visuals per type: alpha (He4 ball), beta (electron), etc.

## Steps
1. In scene, after renderNucleus, for each decay, create product nucleus graphic smaller.
2. Animate decay: shake main, emit product.
3. But for static view first.
4. Position products in row below.
